# POPs-Light
A light version of POPs Jquery plugin 
You can use this plugin to create custom popups 

## Usage

To use it, it's easy :

In the HTML code : 
`<div id="pop"> Hey, it's a Popup</div>
`

```js      
$('#pop1').myPOP({
  "title":"POPs Light",
  "eventObj":$("#launcher1"),
  "color1":"#ABEBC6",//Color of popups'header
  "color2":"#1D8348",
  "width":"800px",
  "height":"300px"
});
```
